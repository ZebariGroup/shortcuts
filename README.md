## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Categories

- **Windows** - Windows OS shortcuts
- **macOS** - macOS shortcuts
- **Browser** - Web browser shortcuts (Chrome, Firefox, Edge)
- **Code** - Code editor shortcuts (VS Code)
